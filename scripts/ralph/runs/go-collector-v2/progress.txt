## Codebase Patterns
- Go binary 位于 packages/logger-collector/，零外部依赖，只用 Go stdlib
- server 包在 internal/server/，负责 HTTP 和启动输出
- query 包在 internal/query/，负责 read/filter/format
- main.go 负责 CLI 路由和参数解析
- Node 薄壳在 packages/logger/src/cli.ts，零外部依赖
- quality-gate 的品牌规范：templates/banner.txt（ASCII art）+ 5 色渐变（38;5;75/105/141/175/204）
- quality-gate 的 ECAM 面板：C 颜色常量 + ╔═╗ box-drawing + dim green 边框 + 内宽 60
