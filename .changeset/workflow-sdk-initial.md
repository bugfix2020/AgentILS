Add `@agent-ils/workflow-sdk`: framework-agnostic workflow engine with sequential node execution, context patching, stop/continue signals, hooks, and AbortSignal support. Includes React (`useWorkflow`) and Vue (`useWorkflow`) bindings with per-node state tracking, and interactive API Key management examples (React+Antd, Vue+ElementPlus).
